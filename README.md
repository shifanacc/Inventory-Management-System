# Inventory Management System
## Introduction
The Inventory Management System is a menu-driven Python mini project developed using Core Python.
The main purpose of this project is to manage product information in an organized and efficient manner.
It allows users to add, view, update, and delete product details through a simple user interface.
This project demonstrates the practical use of basic Python programming concepts.
## Project Description
This system stores product information such as product ID, name, price, and quantity.  
The program continuously displays a menu to the user and performs operations based on the userâ€™s choice until the exit option is selected.
## Features
- Add product details  
- View product list  
- Update product quantity  
- Delete a product  
## Tools used
Python (Core Python)
## How to Run the Project
1. Download the repository
2. Open the project folder
3. Run the Python file using the command:
   python inventory_management_system.py
4. Follow the on-screen menu instructions
